F3. CSS


1. Vad är en regel i CSS och vad består den av?

Svar: En CSS regel är en regel som bestämmer hur ett eller flera element i html-koden ska se ut och består av två delar: en selektor, som visar vilket av elementen i html-dokumentet som ska beröras av regeln och en lista av regler som är uppbyggd av egenskaper och värden. Ett exempel på en CSS regel är "body {background-color:mistyrose ;}" där body är selektorn, background-color är egenskapen och mistyrose är värdet. Regeln bestämmer alltså att bodyn i html-dokumentet ska vara rosa.


2. Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Svar: Skillnaden är att man med id kan märka ut enstaka element i html-koden, medan man med class kan märka ut flera olika element samtidigt. Man bör använda id när man endast vill arbeta med enstaka element i css-koden och class när man vill arbeta med flera element samtidigt. Med class kan man exempelvis välja ut ett antal av samma typ av element som man vill ändra istället för att arbeta med alla element av samma typ i css.
